Tony Hawk's Pro Skater 3

Tony Hawk's Pro Skater 3 is a 2001 skateboarding game that was developed by Neversoft, Edge of Reality for the Nintendo 64 port, Shaba Games for the PlayStation port, Vicarious Visions for the Game Boy Advance port, Gearbox Software for the PC port and HotGen for the Game Boy Color port and was published by Activision. It is the third game in the "Tony Hawk" series.

"Tony Hawk's Pro Skater 3" was the last game ever to be made for the Nintendo 64. Less than 400 copies of the N64 port are known to exist. It was also the first PlayStation 2 game to support online play, predating the actual online hook up.
