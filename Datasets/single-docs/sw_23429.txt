Age of Empires III

Age of Empires III is a 2005 computer game made by the company Ensemble Studios. It is published by Microsoft. It is the third game of the Age of Empires games, and has better graphics than the ones before it. It is a real-time strategy game. The plot is from 1500 to 1860. An expansion pack, "Age of Empires III: The WarChiefs", was released for the game on October 17, 2006.

It was published several years ago (October 18, 2005), so it will run on almost any recent computer. It is only compatible with Microsoft Windows XP and Windows Vista. The game is on three CDs.
