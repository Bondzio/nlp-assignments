Shabu shabu

Today, there are many kinds of shabu-shabu. There is some kinds of Shabushabu. In hokkaido, there is "Takosyabu" which uses octopus. In Nagoya, there is "Torisyabu" which uses Nagoyacotin" In Toyama, there is "Sakesyabu" which uses salmon. In Kagoshima, there is ""Kurobutashabu" which use "Kagoshima black pig".
