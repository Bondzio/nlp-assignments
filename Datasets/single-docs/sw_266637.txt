Calcium permanganate

Calcium permanganate is a chemical compound. Its chemical formula is Ca(MnO). It contains calcium and permanganate ions.

Calcium permanganate is a dark purple solid. It is a powerful oxidizing agent. It is similar to potassium permanganate. It can catch some organic chemicals like alcohol on fire. It can cause an explosion when mixed with sulfuric acid. This is because manganese(VII) oxide. 

It can be made by calcium oxide with manganese(IV) oxide and calcium nitrate. 

It might whiten teeth. It can be used as an oxidizing agent, although potassium permanganate is more common.
