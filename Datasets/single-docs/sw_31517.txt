Game Boy Color

The Game Boy Color is a video game console manufactured by Nintendo. It was first introduced in Japan on October 1998. It is the successor to the original Game Boy, but it precedes the Game Boy Advance. The biggest innovation of the Game Boy Color was its colorful graphics. The Game Boy Color is almost as powerful as the Nintendo Entertainment System. With the exception of the Pokemon Mini, it was the last 8-bit console made by a mainstream company. The last game released for Game Boy Color was "Harry Potter And The Chamber of Secrets" in 2002.
