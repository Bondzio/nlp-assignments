Python molurus

Python molurus is a large nonvenomous python species. It is found in many tropic and subtropic areas of South Asia and Southeast Asia. 

It has various common names: Indian python, black-tailed python and Indian rock python. The species is limited to Southern Asia. It is generally lighter colored than the Burmese python. It can grow to .
