Passbook (application)

Passbook is an application in iOS that lets users to keep coupons, boarding passes, event tickets, store cards, 'generic' cards, and other forms of mobile payment. It was made by Apple and announced at the 2012 Apple Worldwide Developers Conference on June 11, 2012. The application was released with iOS 6 on September 19, 2012.

The Passbook application shows the following kinds of 2D barcodes: Aztec, PDF417, and QR. Each digital coupon or ticket is called a "pass". When the user starts Passbook for the first time, a short introduction screen will appear with a button inviting users to find applications on the App Store that integrate Passbook. Passes can also be distributed with the internet via Safari or sent to another user using e-mail.

Passes are synced between iOS devices using iCloud. OS X 10.8.2 and newer also support opening passes to be sent to users' iOS devices.
The application is available in iOS 6 or later. It is currently only available on iPhone and iPod Touch, but not on iPad.
